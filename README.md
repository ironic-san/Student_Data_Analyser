# Student Data Analysis Project

This project analyzes student performance using Python:
- Cleans missing values with Pandas
- Calculates total marks & averages using NumPy
- Visualizes data using Matplotlib

## Features
- Data cleaning
- Bar chart of average marks
- Scatter plots of attendance vs marks
- Histogram of total marks distribution

## Dataset
student_data.csv (contains marks of 40 students)
